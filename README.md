# Weather App

#### Simple weather app

#### By Scott McIntire

## Description

A user can enter a location and see weather details. 

## Setup Instructions

* Clone this repository.
* Install Node, Bower, Gulp.
* Run the index.html file.

## Technologies Used

Node, Bower, Gulp, JSON, Bootstrap, CSS, JavaScript, HTML
